# Images Directory

Place your PCB design images in this directory.

## Recommended Image Sources

### Free Stock Photos:
- **Unsplash**: https://unsplash.com/s/photos/printed-circuit-board
- **Pexels**: https://www.pexels.com/search/pcb/
- **Pixabay**: https://pixabay.com/images/search/pcb/

### Image Types Needed:
- PCB layouts and designs
- Circuit diagrams
- Component close-ups
- Manufacturing process photos
- Design software screenshots
- Oscilloscope waveforms
- Thermal images

## Image Guidelines:
- **Format**: JPG or PNG
- **Size**: Optimize for web (max 1920px width)
- **Naming**: Use descriptive names (e.g., `pcb-layout-example.jpg`)
- **Alt Text**: Always include alt text in HTML for accessibility

## Current Usage:
Images are referenced in HTML files using:
- Unsplash API for placeholder images
- YouTube thumbnails for videos
- Local images from this directory
